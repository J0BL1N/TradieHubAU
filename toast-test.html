Accessibility Features

  • Auto-dismiss after 5 seconds
  • Respects prefers-reduced-motion
  • ARIA live regions for screen readers
  • Keyboard accessible (close button)
  • Non-blocking UI (doesn't interrupt workflow)

Integration Examples

Job Application Event 

window.ATHToast.show({
  message: 'New application received for Kitchen Renovation!',
  icon: 'briefcase',
  link: { href: 'jobs.html#job-123', label: 'View Job' }
});

Message Received Event 

window.ATHToast.show({
  message: 'New message from Sarah Johnson',
  icon: 'message-circle',
  link: { href: 'messages.html?conversation=conv-456', label: 'View' }
});

Custom Duration

window.ATHToast.show({
  message: 'This stays for 10 seconds',
  icon: 'clock',
  duration: 10000
});
